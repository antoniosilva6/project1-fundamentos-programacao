# 1º projeto de Fundamentos da Programação

Neste primeiro projeto de Fundamentos da Programação os alunos o objetivo foi desenvolver
funções que permitam resolver três tarefas independentes: 1) Justificação de textos;
2) Cálculo de mandatos pelo método de Hondt; e 3) Solução de sistemas de equações.

Ver o documento PDF para mais informações.
